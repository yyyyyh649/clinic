# 图片资源说明

本目录需要放置以下图片资源：

## Tab Bar 图标

底部导航栏图标，建议尺寸 81x81 像素，PNG格式，支持透明背景：

| 文件名 | 说明 |
|--------|------|
| home.png | 首页图标（未选中状态） |
| home-active.png | 首页图标（选中状态） |
| records.png | 档案图标（未选中状态） |
| records-active.png | 档案图标（选中状态） |
| profile.png | 我的图标（未选中状态） |
| profile-active.png | 我的图标（选中状态） |

## 其他图片

| 文件名 | 说明 |
|--------|------|
| default-avatar.png | 默认用户头像 |
| article1.png | 文章封面图1（示例） |
| article2.png | 文章封面图2（示例） |
| article3.png | 文章封面图3（示例） |

## 图标来源建议

1. **阿里巴巴矢量图标库**：https://www.iconfont.cn/
2. **Remix Icon**：https://remixicon.com/
3. **Feather Icons**：https://feathericons.com/

## 临时替代方案

在正式图标准备好之前，小程序将使用 emoji 表情符号作为临时图标。正式上线前请替换为规范的图标文件。

## 注意事项

- 所有图标建议使用统一的设计风格
- 选中状态图标建议使用主题色（#4A90D9）
- 未选中状态图标建议使用灰色（#999999）
